# ESC-Tester

A program to test up to four ESCs at once with an arduino nano

| LED colour | Meaning         |
| ---------- | -------         |
| Red        | Not sending     |
| Purple     | Starting ESCs   |
| Orange     | Minimum signal  |
| Blue       | Neutral signal  |
| Green      | Maximum signal  |
| Grey/White | Non zero signal |
